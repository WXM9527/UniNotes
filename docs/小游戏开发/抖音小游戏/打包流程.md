- [官方文档](https://developer.open-douyin.com/docs/resource/zh-CN/mini-game/guide/unitygame/introduction)



- 核心逻辑

  - 利用官方插件，打包出一个 apk

  

- 工作

  - 发布和测试都需要完善后台帐号
  - [下载插件 BGDT（ByteGame Developer Tools）](https://developer.open-douyin.com/docs/resource/zh-CN/mini-game/develop/guide/game-engine/rd-to-SCgame/BGDT-handbook/install)



- 注意事项

  - 若游戏工程所使用的 Unity 版本不是所支持的版本，则需要先将游戏工程的 Unity 版本改变为所支持的版本。

  - 安全性考虑，**不允许**在项目中加入原生 java 代码库。



- 必须加入以下抖音功能
  - 录屏
  - 分享
